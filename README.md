## Partie personnelle 

On a choisi de faire ce jeu pour tester un peu le développement de ce qu'on appelle les "boomer shooters" tel que Doom, Quake, Wolfenstein, ect. et voir si on pouvait réaliser une démo de jeu dans cette veine. ça fait longtemps que 
je pense à me lancer sérieusement dans la réalisation d'un jeu de ce type sur mon temps libre et j'ai eu l'occasion de découvrir un peu avec l'aide d'un ami. Pour le théme de l'IA on s'est dit qu'étrangement, faire un jeu sur le théme de l'IA telle que celle dont on parle actuellement (GPT, Claude, Autres IA génératives) était peut-être trop évident. Pour rémédier à cela, on a décidé de plûtot partir sur une esthétique et narrative IA en utilisant donc le théme pour notre choix des personnages et de l'ambiance du jeu. Ici, on se bat contre des robots qui mettent toute leur intelligence artificiel au service d'essayer de tuer notre personnage, un cyborg originalement plus chair que métal. Du côté des difficultés, Bien que certains merge mal pensés nous ont certainement donné un peu de fil à retordre, La plus grande difficulté était la recherche et l'usage des modèles d'ennemies, qui était décidément très turbulent dans Blender ET dans le javaScript. Encore maintenant, je pense qu'en les regardant bouger dans le jeu on réalise qu'il y a eu certains abandons sur la qualité de leur déplacement.


# BreakingPointBabylonJS

A compact Babylon.js web game demo featuring enemy AI, projectile combat, and FPS gameplay.

## Synopsis

You're a cyborg fighting robots through a series of rooms. Grab items to enhance your capabilities and keep yourself alive until the level is cleared of all its AI foes.

## Running the project

1. Run "npm run dev" in the root folder terminal

## Project structure

- `index.html` — main HTML entry point
- `js/` — game logic, AI, and map management scripts
- `lib/` — Babylon.js runtime library
- `css/` — basic UI styling
- `models/` — 3D assets used by the game

## Notes

This repository is intended as a lightweight Babylon.js prototype for exploring game AI and physics in the browser.
